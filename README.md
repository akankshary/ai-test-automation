# 🚀 AI Test Automation Framework

An AI-powered test automation framework built using **Playwright, JavaScript, and GitHub Actions**, supporting end-to-end UI testing, API validation, and CI/CD pipelines.

---

## ✨ Features

- ✅ End-to-end UI testing using Playwright  
- ✅ API testing using Axios  
- ✅ Positive & negative test coverage  
- ✅ HTML reports with screenshots, videos, and traces  
- ✅ CI/CD pipeline using GitHub Actions  
- ✅ Scalable test structure and reusable utilities  
- 🤖 AI-assisted test generation for test case ideas  

---

## 🛠 Tech Stack

- JavaScript  
- Playwright  
- Axios  
- Node.js  
- GitHub Actions
